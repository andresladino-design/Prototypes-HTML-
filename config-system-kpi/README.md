# Config System KPI

Prototipo de la **configuración de KPIs** dentro del Operation Center de Simetrik. Permite definir, parametrizar y gestionar los KPIs del sistema que alimentan los monitores.

**Última actualización:** 2026-05-25
**Estado:** 🟢 Activo
**Proyecto:** Operation Center

## Cómo verlo

- **Online:** https://andresladino-design.github.io/Prototypes-HTML-/config-system-kpi/
- **Local:** abrir `index.html` en el navegador

## Stack

- Tailwind CSS (CDN)
- Alpine.js
- Lucide Icons
- desyk tokens
