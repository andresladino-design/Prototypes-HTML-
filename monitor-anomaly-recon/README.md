# Monitor Anomaly Recon

Prototipo del **monitor de anomalías de conciliación** para el Operation Center de Simetrik. Visualiza, prioriza y permite actuar sobre anomalías detectadas en procesos de conciliación.

**Última actualización:** 2026-05-25
**Estado:** 🟢 Activo
**Proyecto:** Anomaly Management (Operation Center v2.8)

## Cómo verlo

- **Online:** https://andresladino-design.github.io/Prototypes-HTML-/monitor-anomaly-recon/
- **Local:** abrir `index.html` en el navegador

## Stack

- Tailwind CSS (CDN)
- Alpine.js
- Lucide Icons
- desyk tokens
